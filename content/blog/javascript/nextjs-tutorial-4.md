# Title 변경

```js
import Head from 'next/head'
```

Head를 import 해준다.

```js
<Head>
  <title>Article List</title>
</Head>
```

컴포넌트 내부에 Head 태그로 감싼 title 태그를 정의 후 원하는 타이틀을 넣는다.

해당 페이지의 title을 재정의 해주게 된다.
